Twitter Sentiment Analytics Dashboard



Institute Lab Project – Big Data Analytics \& Technologies



End-to-end NLP and business intelligence project using a real-world Twitter dataset (2009).



Overview



This project transforms raw tweet data into structured analytical insights using Python and Power BI.



Pipeline:



Data Cleaning → Feature Engineering → 3-Class Sentiment Classification → Aggregated Tables → Interactive Dashboard



Implementation



Text preprocessing and feature engineering (length, word count, URL/mention/hashtag flags)



Sentiment scoring using TextBlob polarity



Threshold-based 3-class classification (Negative / Neutral / Positive)



Fact and summary tables designed for Power BI modeling



KPI measures and trend analysis using DAX



Technologies



Python, Pandas, TextBlob, Power BI, DAX, Git



Skills Demonstrated



NLP preprocessing, sentiment analysis, data modeling, dashboard development, end-to-end analytics workflow.
